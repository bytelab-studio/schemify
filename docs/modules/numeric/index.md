# Numeric Module

The **numeric** module provides validators for advance integers types. For example to validate that a number is in
bounds of an `uint16`.

## Validators

- [`Schemify.int8`](/modules/numeric/int8)
- [`Schemify.int16`](/modules/numeric/int8)
- [`Schemify.int32`](/modules/numeric/int8)
- [`Schemify.int64`](/modules/numeric/int8)
- [`Schemify.numeric`](/modules/numeric/numeric)
- [`Schemify.uint8`](/modules/numeric/numeric)
- [`Schemify.uint16`](/modules/numeric/numeric)
- [`Schemify.uint32`](/modules/numeric/numeric)
- [`Schemify.uint64`](/modules/numeric/numeric)
